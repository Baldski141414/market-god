# 🔮 Market God

Live market intelligence dashboard with real-time prices, latest news, and AI-powered buy/sell/hold signals.

## Assets Tracked
- **NVDA** - NVIDIA Corporation
- **TSLA** - Tesla Inc
- **AAPL** - Apple Inc
- **MSFT** - Microsoft Corporation
- **BTC** - Bitcoin

## Features
- 🔴 Live price data via Yahoo Finance API
- 📰 Latest news headlines per asset
- 🤖 AI-powered Buy/Sell/Hold signals with confidence scores
- 📊 Price change indicators & portfolio allocation suggestions
- 🌙 Dark mode, responsive design
- ⚡ Auto-refresh every 60 seconds

## Stack
- **Frontend**: Vanilla HTML/CSS/JS (zero dependencies, instant load)
- **Data**: Yahoo Finance API (via allorigins CORS proxy)
- **News**: Embedded from scraped analysis

## Run Locally
```bash
# Just open index.html in your browser — no build step needed
open index.html
```

## Deploy
Drop `index.html` onto any static host: Vercel, Netlify, GitHub Pages.
